- In CDD_RegulatorManager.c, all the runtime checks like `if ((boolean)TRUE == Get_pLMMFuncEnable)` will evaluate false, so LMM-specific actions such as `CDD_RegMngr_LMM_PreInit()`, `CDD_RegMngr_LMM_Init()`, `CDD_RegMngr_LMM_DeInit()`, `tps9266x_main()`, `tps9266x_force_deinit()`, LMM diagnosis, and the LMM supply-disable request path will not run at runtime.
